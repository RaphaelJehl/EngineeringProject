Comment installer la base de données et se connecter au site : 
- Télécharger et installer Node JS et Mongo DB

- Démarrer le serveur Mongo DB en exécutant la commande suivante dans le dossier "bin" ou Mongo DB est installé :
    > ./mongod
    ou
    > mongod

- Démarrer une invite de commandes dans le dossier ou se trouve le fichier "server.js" et exécuter les commandes suivantes :
    > npm update
    > npm install -g nodemon
    > nodemon server.js

- Ouvrez votre navigateur et entrez l'adresse suivante :
    > http://localhost:3000/

Si vous rencontrez un quelconque problème, n'hésitez pas à nous contacter.
